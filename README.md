# vuecanvasmap（开发中。。）

通用化地图解决方案。采用canvas覆盖物的方式使用WebGL，Canvas进行绘画。从而可实现一套坐标系基于多种底图地图的操作。分支实验了其他种可能性，来研究探讨出最佳方案。
